Detailed Description

Pools pre-allocate space for objects in the game to quickly allocate and de-allocate individual objects.

Objects in pool must contain members:

int32_t p_Pool_Prev;
int32_t p_Pool_Next;
int32_t p_Pool_InUse;
int32_t p_Map;
